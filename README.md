# Data Science Assignments

A comprehensive collection of **20 Data Science assignments** covering fundamental to advanced topics in Machine Learning, Statistics, and Data Analysis.

---

## Quick Start

### Installation
```bash
pip install -r requirements.txt
```

### Running Notebooks
Each folder contains a Jupyter notebook (`.ipynb`) with complete solutions. Open them using:
```bash
jupyter notebook
```
